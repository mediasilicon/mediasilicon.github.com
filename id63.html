IMA ADPCM-SO1 ASIC/FPGA core

1. Important Features

  • 4K gates size
  • Low power dissipation, required encoding clock equal to audio sampling rate
  • Single clock synchronous design
  • Support 4 bits IMA/DVI ADPCM coding and decoding
  • Straightforward interface for SoC applications

2.Function Overview

IMAADPCM-SO1 takes any 16 bits PCM bit stream, automatically compresses the PCM stream to 4 bits IMA ADPCM stream.  The IMAADPCM-SO1 can also decode the compressed 4 bits IMA ADPCM stream to 16 bits PCM bit stream. It is an ideal hardwire block for voice communication and recoding SOCs.

3.Interfaces

4.Applications

IMAADPCM-SO1 can be integrated with Media Silicon’s MP3-SO3 MP3 decoder AISC core for a complete hardwire MP3 Player/Voice recoding device or  used for other voice recoding and communication devices.
